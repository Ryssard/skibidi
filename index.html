element blokowy main- okresla blok glowny
element blokowy aside - okresla blok boczny
element blokowy nav - okresla blok nawigacji (menu)
element blokowy footer - kresla stopke (dol) strony
element blokowy section - ogolny element zgodny z html5
element blokowy div - elemnet uzywany w starszych wersiach html (przed html5), nie zaleca sie jego stosowania w kodzie

element blokowy paragraf(akapit) "p" - w nim umieszcza fragment tekstu w ksiazce, po zakonczeniu akapitu tekst rozpoczyna sie od nowej liniki
